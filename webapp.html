4 Dataset

ช่วงเลขตามกลุ่ม เช่น ๒๑–๒๙ (9 classes)
Dataset สำหรับ train: อย่างน้อย 30–50 รูป / class
ต้องสร้าง dataset เอง โดยอาจพัฒนา web สำหรับเก็บข้อมูล
โปรเจกต์นี้ใช้ dataset ลายมือจริง ~388 รูป/หลัก จาก Thai Handwriting Number Dataset

5 Backend

Flask หรือ FastAPI
/predict endpoint — รับภาพ base64 ส่งกลับผลลัพธ์
/upload-model endpoint — hot-swap model ไม่ต้อง restart
/model-info endpoint — ดูสถานะ model ปัจจุบัน

6 Preprocessing

Grayscale → Invert สี (canvas พื้นขาว → model พื้นดำ)
Auto-crop bounding box → center ใน canvas
Resize เป็น 64×64 px
Normalize ค่า pixel เป็น 0–1
Augmentation: Random Rotation, Translation, Zoom, Blur, Noise

9 คำแนะนำ — การสร้างระบบเก็บข้อมูล (Optional)

ส่วนนี้เป็น Optional: เป็นแนวทางสำหรับกลุ่มที่ต้องการสร้าง dataset อย่างเป็นระบบ
แนวคิด: ใช้หน้าเว็บที่มี canvas เป็นเครื่องมือเก็บข้อมูล มีปุ่ม Save Sample
หน้าจอ: dropdown เลือก label + canvas + ปุ่ม Save + ปุ่ม Clear
Flow: เลือก label → เขียนตัวเลข → กด Save → Clear → ซ้ำ
Backend: เพิ่ม endpoint /save-sample รับ image + label
รูปแบบ: เก็บเป็น PNG แยกโฟลเดอร์ตาม class เช่น dataset/๒๑/
ชื่อไฟล์: 21_001.png, 22_001.png, ...
Train/Test split: แยก 80/20 หรือ 70/30 อย่างชัดเจน
ข้อดี: dataset ตรงกับ input จริง เพราะมาจาก canvas แบบเดียวกัน
